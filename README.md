# TomideStreams SDK

The TomideStreams TypeScript SDK provides a robust interface to interact with a movie database API, facilitating operations like fetching random movies, searching for movies based on keywords, and retrieving detailed information about movies using their IMDb ID.

## Installation

To install the SDK, use the following npm command. The SDK relies on Axios for making HTTP requests, so ensure Axios is installed and configured in your project.
